# Doctor Appointment Management System

A full-stack web application designed to simplify the process of booking and managing doctor appointments. The system provides an easy-to-use interface for patients to book appointments and for doctors to manage their schedules efficiently.

---

## Problem Statement

In many hospitals and clinics, appointment booking is still handled manually or through inefficient systems, leading to long waiting times, scheduling conflicts, and poor patient experience. This project aims to digitize and streamline the appointment process.

---

## Solution Overview

The Doctor Appointment Management System allows patients to view available doctors and time slots, book appointments online, and receive confirmation. Doctors can manage their availability, view upcoming appointments, and maintain an organized schedule through a dedicated dashboard.

---

## Key Features

- Patient appointment booking system  
- Doctor availability and slot management  
- Separate dashboards for doctors and patients  
- Real-time appointment updates  
- Secure authentication and authorization  
- Responsive and user-friendly UI  

---

## Tech Stack Used

### Frontend
- React / Next.js  
- Tailwind CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

---

## Application Workflow

1. Patient registers or logs in to the system  
2. Patient views available doctors and time slots  
3. Appointment is booked for a selected date and time  
4. Doctor can view and manage booked appointments  
5. Doctor updates availability as needed
